# Календарь (PWA)

## Как обновить/установить
1. Залей содержимое этой папки в **корень** репозитория GitHub.
2. Settings → Pages → Deploy from a branch → `main` + `/(root)`.
3. Открой сайт в Safari и добавь на экран «Домой»:
   Поделиться → На экран «Домой»

## Иконка
По умолчанию используется вариант с красными двумя последними ячейками (#ffaea8).
Если хочешь вернуться к полностью белой сетке — поменяй ссылки в `manifest.json` и в `<head>` `index.html`
на файлы без `-redweekend`.
